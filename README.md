# Notebook Manager 📓

Un'app responsive in React per gestire quaderni digitali e note. L'ho creata per fare pratica con la gestione dello stato, i Portals e il layout adattivo con Tailwind.

### Tech:
* **React** (Vite)
* **Tailwind CSS**
* **React Portals** (per il modal di errore)

### Come si usa:
1. `git clone` del repository
2. `npm install`
3. `npm run dev`

### Funzioni principali:
* **Gestione Notebook**: creazione e cancellazione di quaderni con titolo, data e descrizione.
* **Note rapide**: aggiunta e rimozione di note all'interno di ogni quaderno.
* **UI Responsive**: sidebar laterale su desktop che si sposta in alto su mobile per ottimizzare lo spazio.
